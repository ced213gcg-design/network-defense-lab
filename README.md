# Network Defense Lab

This repository documents hands-on exercises focused on defending networks from suspicious activity and potential cyber threats.

## Objectives

• Understand how networks are monitored for threats  
• Identify suspicious traffic patterns  
• Practice defensive security thinking  
• Learn how attackers probe networks

## Skills Practiced

• Network traffic analysis  
• Identifying port scans  
• Recognizing suspicious IP activity  
• Understanding firewall and IDS alerts

## Lab Exercises

### Traffic Inspection
Analyzing simulated network traffic to detect abnormal behavior.

### Port Scan Identification
Recognizing patterns generated by common scanning techniques.

### Suspicious Connection Analysis
Investigating unusual network communication attempts.

### Defensive Monitoring
Learning how defenders observe and respond to network anomalies.

## Purpose

This project demonstrates foundational network defense skills as part of my cybersecurity learning journey and professional portfolio.
